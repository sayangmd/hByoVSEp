# 前言

欢迎来到我们的开源项目——基于微信小程序的学习资料销售平台。本项目是一款集在线学习、资料购买于一体的教育类小程序，致力于为广大用户提供优质的学习资源。接下来，让我们一起了解一下本项目的内容、技术以及如何获取源码等。

# 内容介绍

本项目主要功能包括：用户注册登录、学习资料分类浏览、搜索、购买、在线支付、学习进度跟踪等。通过微信小程序，用户可以随时随地查看和购买所需的学习资料，满足不同场景下的学习需求。后台管理系统则负责资料上传、订单管理、用户管理等功能，方便管理员进行运营管理。

# 技术介绍

本项目采用以下技术栈：

## 语言：Java

## 使用框架：Spring、SpringMVC、MyBatis、微信小程序

## 前端技术：JS、Vue、CSS3、Uniapp

## 开发工具：IDEA/Eclipse、Uniapp

## 数据库：MySQL 5.7/8.0

## 数据库管理工具：phpStudy/Navicat

## JDK版本：jdk1.8

## Maven：apache-maven 3.8.1-bin

## 前端环境：Node.Js 12\14\16
